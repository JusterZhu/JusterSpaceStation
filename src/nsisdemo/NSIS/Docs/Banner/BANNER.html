BANNER插件

该插件显示一个带有可定制文本的横幅。它使用UI的IDD_VERIFY对话框。

有三个函数 - show、getWindow和destroy。

使用方法

Banner::show "要显示的文本"

[可选] Banner::getWindow

Banner::destroy

请参阅Example.nsi获取示例。

Modern UI

现代UI在IDD_VERIFY对话框上有两个标签。要更改所有文本，请使用：

Banner::show /set 76 "文本1（替换为请等待，直到安装加载...）" "普通文本"

自定义UI

如果您的IDD_VERIFY对话框上有更多标签，可以使用多个/set参数来更改文本。

例如：

Banner::show /set 76 "bah #1" /set 54 "bah #2" "普通文本"

/set的第二个参数是控件的ID。

更多技巧

如果您使用/set来设置主字符串（IDC_STR，1030），则可以为窗口的标题和主字符串指定不同的字符串。

如果您将空字符串用作主字符串（Banner::show“”），则横幅窗口将不会显示在任务栏上。
